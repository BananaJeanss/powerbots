A simple, yet powerful discord bot.

  • Github Repository
  • Invite the bot
  • Testing server

Features

  • All the moderation tools you need, ranging from notes to bans.
  • Image manipulation commands, such as deepfry, pixelate, and togif.
  • Basic utility commands such as serverinfo, whois, and more.
  • And other commands such as 8ball, coinflip, github, npm, dictionary, etc.

Basic commands

Command Description
/help Shows a list of available commands.
/serverinfo Displays information about the server.
/whois Shows information about a user.
/8ball Answers a yes/no question.
/coinflip Flips a coin.
/github Get information about a user, or a repo.
/npm Get information about a package.
/deepfry Deepfry an image, self explanatory.
/pixelate Pixelate an image.

More commands can be found via commands.md.
